# Ionic 4

El proyecto utiliza Ionic [Ionic Framework](https://ionicframework.com/) version 4

### Pre instalación
Verificar versión de ionic
```sh
$ ionic -v
```
#### Instalación (OSx):
```sh
$ git clone https://github.com/nacho-test-developer/ionic4-tab.git
$ cd ionic4-tab
$ npm i
$ npm start
```
